# 교대학식

서울교육대학교 교내 학생식당의 주간 식단 정보를 제공하는 iOS 앱입니다.

## 주요 기능

### 오늘 탭
- 오늘과 내일의 중식·석식 메뉴를 카드 형태로 표시
- 메뉴 항목별 가격 표시
- 식단 데이터 없는 날(주말 등)에 대한 빈 상태 처리

### 주간 탭
- 이번 주(월~금) 전체 식단 표시
- 주차 이동(이전/다음 주) 지원
- 달력으로 원하는 주차 직접 선택
- 데이터 없는 날짜는 달력에서 선택 불가 처리

### 알림 기능
- **중식 알림**: 매일 지정한 시각에 로컬 푸시 알림 (기본 11:30)
- **석식 알림**: 매일 지정한 시각에 로컬 푸시 알림 (기본 17:30)
- **주간 식단 업데이트 알림**: 매주 월요일 오전 9시 알림
- 시간 피커로 알림 시각 변경 가능
- 알림 권한 안내 모달 (최초 요청 / 거부 시 설정 앱 유도)
- 알림 권한 해제 시 앱 내 알림 설정 자동 OFF 동기화
- 알림 탭 시 앱의 오늘 탭으로 자동 이동

### 설정 탭
- 알림 설정 (중식·석식·주간 알림 ON/OFF, 시각 변경)
- 오픈소스 라이선스 목록

### 데이터 동기화
- Firebase Firestore에서 식단 데이터 수신
- SwiftData로 로컬 캐싱, 네트워크 없이도 최근 데이터 사용 가능
- 앱 포그라운드 진입 시 자동 동기화 (날짜 변경 감지 포함)
- 네트워크 미연결 시 오류 알림 표시

## 기술 스택

| 분류 | 사용 기술 |
|------|-----------|
| 언어 | Swift 6 |
| UI | SwiftUI |
| 로컬 저장 | SwiftData |
| 백엔드 | Firebase Firestore |
| 알림 | UserNotifications (로컬 알림) |
| 네트워크 | Network.framework (NWPathMonitor) |
| 최소 배포 타깃 | iOS 26 |
